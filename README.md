# HPIndel-Profiler
Command-line tool for detecting and quantifying indel events in homopolymer and non-homopolymer regions from BAM files using BED-defined genomic intervals. By default the repo has EZH2 9bp Homopolymer region Chr7:148526848 with 50bp flanks.
